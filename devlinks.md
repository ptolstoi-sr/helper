### DB User
```
CREATE DATABASE '<db>';
CREATE USER '<dbuser>'@% IDENTIFIED BY '<pwd>'; GRANT USAGE ON '<db>'.* TO '<dbuser>'@'%' IDENTIFIED BY '<pwd>'; 
GRANT ALL ON '<db>'.* TO '<dbuser>'@'%' IDENTIFIED BY '<pwd>'; SHOW GRANTS FOR '<dbuser>';
```
